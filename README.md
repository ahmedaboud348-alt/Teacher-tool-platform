# Teacher Tools Platform

منصة أدوات للمعلمين مبنية بـ Next.js، تساعد على إنشاء وتصدير أوراق الامتحانات.

## المميزات

- إنشاء أوراق امتحانات مخصصة حسب المستوى والمسار
- حساب توزيع النقاط تلقائياً
- تصدير الأوراق بصيغة PDF أو Word
- دعم مواد الفيزياء والكيمياء

## التقنيات المستخدمة

- [Next.js](https://nextjs.org/) — إطار العمل الأساسي
- TypeScript — لضمان جودة الكود
- Tailwind CSS — للتصميم

## كيفية التشغيل

```bash
npm install
npm run dev
```

ثم افتح [http://localhost:3000](http://localhost:3000) في المتصفح.
